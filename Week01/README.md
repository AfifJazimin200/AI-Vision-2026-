# Week 1 - Git and GitHub

## Learning Outcomes
- Create a GitHub repository
- Create a branch
- Compare branches
- Add a collaborator

## Practical Activities
- Repository created
- Branch created
- Collaborator added
- README updated

## Reflection
Today, I learned the foundational concepts of version control using Git and GitHub. I successfully created a GitHub repository and practiced branching to manage different versions of my code safely. Additionally, I learned how to compare branches and understood the process of adding a collaborator to a project for team collaboration. This practical session helped me understand how developers track changes and work together effectively in real-world software development.
